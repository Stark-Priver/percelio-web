## Logo

- **Primary Logo:** Orange Pacelio wordmark + icon (PNG, SVG)
- **Secondary:** White version for dark backgrounds
- **Icon Only:** Orange shipping icon for app icon, favicon
- **Minimum Clear Space:** 1x logo height on all sides
- **Do Not:** Stretch, recolor, add effects, place on busy backgrounds

| Attribute | Description |
