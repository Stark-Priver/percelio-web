# 🎬 Video Script — Teaser / Trailer (Launch Campaign)

**Title:** "Something Big is Coming"  
**Length:** 15–30 seconds  
**Style:** Cinematic, mysterious, hype-building  
**Tone:** Bold, exciting, aspirational

---

## 15-Second Version

| Time | Visual | Text on Screen |
|---|---|---|
| **0:00–0:03** | Fast montage: Packages moving, motorcycles, bustling markets, phones buzzing | |
| **0:03–0:07** | Orange Pacelio logo pulses on dark background | "SMARTER DELIVERIES" |
| **0:07–0:11** | Map of Mbeya with delivery dots appearing one by one | "FOR GROWING BUSINESSES" |
| **0:11–0:15** | Pacelio logo full screen + "Coming Soon" | "PACELIO" |

**Audio:** Bass drop, rising tension, then silence + logo thud

---

## 30-Second Version

| Time | Visual | Text on Screen |
|---|---|---|
| **0:00–0:05** | Slow motion: A package being handed over, a motorcycle riding through Mbeya streets at golden hour | |
| **0:05–0:10** | Quick cuts: Shop owner looking hopeful, rider checking phone, customer smiling | |
| **0:10–0:15** | Phone screen showing: "Track your delivery" → Map with moving GPS dot | |
| **0:15–0:20** | QR code scan → "✓ Delivery Confirmed" | "NO MORE GUESSING" |
| **0:20–0:25** | Montage speeds up: Multiple deliveries happening across the city | "NO MORE WAITING" |
| **0:25–0:30** | Logo appears. Text: "Mbeya. June 2026." | "PACELIO IS HERE" |

**Audio:** Building cinematic orchestral + electronic beat, culminating in logo reveal sound

---

## Production Notes

| Element | Detail |
|---|---|
| **Style** | Cinematic, high-energy |
| **Locations** | Mbeya streets, markets, shops |
